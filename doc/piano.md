## 定音
左手定音，右手跟进
```
1. 左手5，2指之间相隔六个键（无论哪个调）
2. 右手1指与左手5指是同一个键的高8度
3. 右手5指与左手2指是同一个键的高8度
4. 右手3指决定大调，或小调音阶
5. 1，3指之间隔两键为小调
6. 1，3指之间隔三键为大调
```
